# L3ABOUCH SEARCHER

Educational Search Helper for Linux and Termux

## Features

* Terminal-based interface
* Kali Linux support
* Termux support
* Multi-platform search shortcuts
* Configurable settings via config.json
* Search history and logging support
* Lightweight and easy to use

## Installation

### Kali Linux / Debian / Ubuntu

```bash
git clone https://github.com/l3abouch/L3ABOUCH-Searcher.git
cd L3ABOUCH-Searcher
python3 l3abouch.py
```

### Termux

```bash
pkg update
pkg install git python
git clone https://github.com/l3abouch/L3ABOUCH-Searcher.git
cd L3ABOUCH-Searcher
python l3abouch.py
```
